Featured Project

Fast Graph Learning Algorithms for NLP Applications

(in progress)

Developing a novel graph learning algorithm to efficiently process large, sparse semantic graphs for natural language processing. Existing techniques are computationally intensive on large graphs. This project aims to exploit sparsity to avoid unnecessary computations and allow graphs with tens of thousands of nodes to be processed quickly. This could enable capturing finer-grained semantic relationships between words and improve natural language understanding. The algorithm design and evaluation is in progress. The project requires expertise in graph signal processing, natural language processing, and algorithm design. It has potential applications in semantic search, document classification, and question answering.
